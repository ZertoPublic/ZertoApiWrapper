# Disconnect-ZertoServer

## SYNOPSIS
Disconnects the current session from the ZVM

## SYNTAX

```
Disconnect-ZertoServer
```

## DESCRIPTION
Disconnects the current session from the Zerto Virtual Manager

## EXAMPLES

### Example 1
```powershell
PS C:\> Disconnect-ZertoServer
```

Disconnects from the Zerto Server
