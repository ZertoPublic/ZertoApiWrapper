# Get-ZertoUnprotectedVm

## SYNOPSIS
Returns all virtual machines at the site not currently protected in a virtual protection group.

## SYNTAX

```
Get-ZertoUnprotectedVm [<CommonParameters>]
```

## DESCRIPTION
Returns all virtual machines at the site not currently protected in a virtual protection group.

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-ZertoUnprotectedVm
```

Returns all virtual machines at the site not currently protected in a virtual protection group.
